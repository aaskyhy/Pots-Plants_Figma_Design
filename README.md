# Pots-Plants_Figma_Design
Pots and Plants – UI/UX Design 
https://www.figma.com/design/9X9B1vk7cliMcBGCn4EE64/Pots-and-Plants?t=qw7FoNzDKN6VMpxn-1

Pots and Plants is a mobile application UI/UX design created in Figma as part of a university mobile application development subject personal project. The application is designed to help indoor plant owners care for their plants by providing easy-to-follow care guides and essential plant information in a user-friendly interface.

Project Objectives:
-Design an intuitive and user-friendly mobile interface.
-Help users access plant care information easily.
-Create a clean and modern design suitable for both beginners and experienced plant owners.
-Plan the application's user flow before development.
